# SegaController
Add support of Sega Mega Drive controller to Linux PC

## Controller pinout

![image_1](https://raw.githubusercontent.com/lxxxxl/SegaController/master/img/pinout.png?raw=true)

__Pin  Button (SEL=5V)__  
__D0__   D-pad up  
__D1__   D-pad down  
__D2__   D-pad left  
__D3__   D-pad right  
__D4__   B button  
__D5__   C button  
 
---------------  
__Pin  Button (SEL=0V)__  
__D0__   up  
__D1__   down  
__D2__   GND  
__D3__   GND  
__D4__   A button  
__D5__   Start button  

---------------  
  
__Pin  Button (SEL=5V after 3rd pulse)__  
__D0__   Z button  
__D1__   Y button  
__D2__   X button  
__D3__   Mode button
